How To Play?
  • User must enter a target score.
  • To start the game click on play.
  • User can move use the arrow keys to select one among three options i.e. Rock, Paper, Scissor.
  • If user selects rock and system randomly selects scissor then user gets one point or if system selects paper then system gets one point.
  • If user first reaches the winning target score then user wins the game otherwise user lose the game.
