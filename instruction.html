How To Play?

1. User must enter a target score.

2. To start the game click on play.

3. User can move use the arrow keys to select one among three options i.e. Rock, Paper, Scissor.

4. If user selects rock and system randomly selects scissor then user gets one point or if system selects paper then system gets one point.

5. If user first reaches the winning target score then user wins the game otherwise user lose the game.
